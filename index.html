Technologies used: HTML, CSS

College Management Website

For this project, I designed a college management system website that allows administrators, teachers, and students to manage different aspects of college activities. This was one of my first attempts at creating a dynamic website, integrating forms for course registration, student attendance tracking, and more.

Technologies used: HTML, CSS, Basic JavaScript

Key features:

  • Login system for different user roles (teacher, student)
  • Course registration and student enrollment management
  • Attendance tracking with reports for teachers
  • Interactive user dashboard for students and staff

This project provided me with hands-on experience in both frontend and backend development, as well as understanding how to handle user authentication, database connections, and dynamic content generation.
